=============================================================
Hyphen Dictionary
=============================================================

This extension allows editors to define a dictionary of words
and optionally word breaks for each of this words.

The extension renders the optional word breaks on the servers side
by adding `&shy;` to the given position.

Why to use this extension?
--------------------------

This extension allows editors to build up a dictionary of words and
optional word breaks that can be used all over the page.

Using a ViewHelper, the integrator can decide where to parse a text through
the hyphen dictionary.
